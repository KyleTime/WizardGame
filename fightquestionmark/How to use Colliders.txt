BoxCol is a box collider
CirCol is a circle collider

checkCol(Collider) < returns a bool of true or false if colliding

example:

CirCol c = new CirCol (float x, float y, float radius);
BoxCol b = new BoxCol(float x, float y, float xSize, float ySize);

if(c.checkCol(b))
{
     //Code for collision
}
